# package in kotlin
## Basic structure

    package org.example
    
    fun printMessage() { /*...*/ }
    class Message { /*...*/ }
    
    // ...

## Example
To create a package named org.example.

One can write these code in .kt file.

    package org.example
    
    fun printMessage() { /*...*/ }
    class Message { /*...*/ }

    // ...

In this file, package name is org.example. 

It has one function printMessage().

It has one class Message.
